Как установить Zsh? ​

IMPORTANT

Многие CLI инструменты, которые мы разрабатываем, требуют ZSH, поэтому установка Zsh является обязательной.

Требования ​

Для Zsh требуется UNIX-совместимое окружение, поэтому на Windows сначала необходимо установить WSL. В этом вам поможет руководство: Как установить WSL?.

На других операционных системах предварительная установка WSL не требуется.

Выполните установку Zsh ​

Zsh на macOS уже должен быть установлен по-умолчанию, поэтому выполнять установку на macOS не требуется.

IMPORTANT

Zsh разрабатывался для UNIX-подобных систем, поэтому на Windows производить установку Zsh необходимо через WSL (Руководство по установке WSL).

Для установки выполните по очереди каждую команду:

shell
sudo apt update
sudo apt install zsh
Установка не требуется
Установка не требуется

Проверьте корректность установки Zsh ​

После завершения установки, проверьте, что Zsh установлен корректно, выполнив следующую команду:

shell
zsh --version

Если команда вывела номер версии, значит Zsh установлен корректно:

shell
zsh 5.9 (arm64-apple-darwin24.0)
