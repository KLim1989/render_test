Какие команды доступны в рабочем пространстве? ​

Все доступные команды описаны в файле melos.yaml.

Как запустить? ​

Для этого выполните melos run [команда] [аргументы команды], например:

shell
melos run pub--get

Подробнее читайте в документации Melos.

Какие команды доступны? ​

build-runner--build ​

Запускает build_runner build --delete-conflicting-outputs.

build-runner--clean ​

Запускает build_runner clean.

build-runner--watch ​

Запускает build_runner watch --delete-conflicting-outputs.

dart--format-code ​

Запускает форматирование кода, написанного на языке Dart.

dart--analyze-code ​

Запускает анализ кода, написанного на языке Dart.

envied-generator--run-for-development ​

Генерирует Dart код на основе файлов [package]/config/app/dev.env.

envied-generator--run-for-production ​

Генерирует Dart код на основе файлов [package]/config/app/production.env.

envied-generator--run-for-regress ​

Генерирует Dart код на основе файлов [package]/config/app/regress.env.

firebase-configurator--run-for-android-development ​

Перемещает файл конфигурации Firebase google-services-development.json из директории [package]/config/firebase/android в директорию, которую отслеживает Firebase.

firebase-configurator--run-for-android-production ​

Перемещает файл конфигурации Firebase google-services-production.json из директории [package]/config/firebase/android в директорию, которую отслеживает Firebase.

firebase-configurator--run-for-ios-development ​

Перемещает файл конфигурации Firebase GoogleService-Info-Development.plist из директории [package]/config/firebase/ios в директорию, которую отслеживает Firebase.

firebase-configurator--run-for-ios-production ​

Перемещает файл конфигурации Firebase GoogleService-Info-Production.plist из директории [package]/config/firebase/ios в директорию, которую отслеживает Firebase.

flutter--clean ​

Запускает flutter clean в выбранных вами пакетах.

intl-utils--generate ​

Генерирует Dart код локализации на основе .arb файлов.

pub--get ​

Запускает flutter pub get.

pub--outdated ​

Запускает dart pub outdated --color.

pub--package-version-print ​

Отображает версию пакета, указанную в поле version в файле pubspec.yaml в формате x.x.x. Всё, что указано в поле version после + будет отброшено.

pub--upgrade ​

Запускает dart pub upgrade --color.

website--build ​

Производит сборку сайта и сохраняет файлы в директории co_website_app/production.

website--deploy ​

Производит сборку сайта и публикует на хостинге.

website--serve ​

Запускает на локальной машине сервер, обслуживающий сайт, в режиме разработки. Такой режим подходит для внесения изменений в сайт и просмотра их в реальном времени.

workspace--clean ​

Удаляет в рабочем пространстве:

  1. Все пустые директории.
  2. Все директории .dart_tool.
  3. Все файлы .flutter-plugins.
  4. Все файлы *.freezed.dart.
  5. Все файлы *.g.dart.

workspace--generate-code-for-development ​

Генерирует код в выбранных пакетах рабочего пространства, используя конфиг [package]/config/app/env/development.env.

workspace--generate-code-for-regress ​

Генерирует код в выбранных пакетах рабочего пространства, используя конфиг [package]/config/app/env/regress.env.

workspace--generate-code-for-production ​

Генерирует код в выбранных пакетах рабочего пространства, используя конфиг [package]/config/app/env/production.env.

workspace--install-packages ​

Устанавливает выбранные пакеты глобально в систему.

workspace--set-up-for-development ​

Подготавливает выбранные пакеты рабочего пространства к работе, используя конфиг [package]/config/app/env/development.env.

workspace--set-up-for-production ​

Подготавливает выбранные пакеты рабочего пространства к работе, используя конфиг [package]/config/app/env/production.env.

yangi-business-app--set-up-for-development ​

Подготавливает пакет yangi_business_app к работе, используя конфиг [package]/config/app/env/development.env.

yangi-finance-app--set-up-for-development ​

Подготавливает пакет yangi_finance_app к работе, используя конфиг [package]/config/app/env/development.env.
